Installation
^^^^^^^^^^^^
Install the latest release of `scirpy` from [PyPI](https://pypi.org/project/scirpy/):

```
pip install scirpy
```

Alternatively, install the latest development version
```
git clone git@github.com:icbi-lab/scirpy.git
cd scirpy
pip install flit
flit install
```
