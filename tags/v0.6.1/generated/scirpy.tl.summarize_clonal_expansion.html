scirpy.tl.summarize_clonal_expansion¶

scirpy.tl.summarize_clonal_expansion(adata, groupby, *, target_col='clonotype', summarize_by='cell', normalize=False, **kwargs)¶

Summarizes clonal expansion by a grouping variable.

Removes all entries with NaN in target_col prior to summarization.

Parameters
adata : AnnDataAnnData

annotated data matrix

groupby : strstr

summarize by this column in adata.obs.

target_col : strstr (default: 'clonotype')

column in obs which holds the clonotype information

summarize_by : {‘cell’, ‘clonotype’}Literal[‘cell’, ‘clonotype’] (default: 'cell')

Can be either cell to count cells belonging to a clonotype (the default), or “clonotype” to count clonotypes. The former leads to a over-representation of expanded clonotypes but better represents the fraction of expanded cells.

normalize : boolbool (default: False)

If False, plot absolute cell counts. If True, scale each group to 1.

**kwargs

Additional arguments passed to clonal_expansion().

Return type

DataFrameDataFrame

Returns

A DataFrame with one row for each unique value in groupby.
