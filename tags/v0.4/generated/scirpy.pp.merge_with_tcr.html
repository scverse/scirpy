scirpy.pp.merge_with_tcr¶

scirpy.pp.merge_with_tcr(adata, adata_tcr, *, how='left', on=None, left_index=True, right_index=True, validate='one_to_one', **kwargs)¶

Merge TCR data with transcriptomics data into a single AnnData object.

Reading in TCR data results in an AnnData object with TCR information stored in obs. Use this function to merge it with another AnnData which contains transcriptomics data.

Will keep all objects (e.g. neighbors, umap) from adata and integrate obs from adata_tcr into adata. Everything other than .obs from adata_tcr will be discarded.

This function uses pandas.merge() to join the two .obs data frames.

Modifies adata inplace.
